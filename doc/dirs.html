FIDOGATE Directories

This document describes the new directory structure for FIDOGATE 4.3.0. (This is to some extend different from 4.2.x!)

The directory / file names listed here are the default values for FIDOGATE 4.3+ source and binary distribution on Unix-like systems.

/etc/fido/
Configuration files live here:
common.conf
FIDOGATE common configuration
gate.conf
FIDOGATE gateway configuration
toss.conf
FIDOGATE tosser configuration
ffx.conf
FIDOGATE ffx configuration
ftp.conf
FIDOGATE ftp hub configuration
aliases
FIDOGATE aliases
areas
FIDOGATE areas - newsgroups conversion
hosts
FIDOGATE hosts
packing
FIDOGATE packing configuration
passwd
FIDOGATE passwords
routing
FIDOGATE routing configuration

/var/lib/fido/
Variable files live here:
seq/
Directory for sequencer files
areas.bbs
Echo areas
fareas.bbs
File areas
history.*
Dupe history data base
Nodelist
Compiled nodelist + index

/var/lock/fido/
Lock files for FIDOGATE programs and scripts live here.

/var/spool/fido/
Fido related programs spool area:
outpkt/
Directory for output FTN packets (.pkt)
mail/
Mail packets created by rfc2ftn
news/
News packets created by rfc2ftn
outrfc/
Directory for output RFC messages (.rfc)
mail/
Mail messages created by ftn2rfc
news/
News messages/batches created by ftn2rfc
toss/
Tosser work directories
toss/
Temporary packets created by ftntoss
route/
Temporary packets created by ftnroute
pack/
Temporary packets created by ftnpack

/var/spool/bt/
Fido BinkleyTerm-style inbound / outbound area:
in/
Insecure inbound directory
pin/
Secure inbound directory (ftntoss default)
uuin/
recvuu inbound directory
out/
Zone 2 outbound directory
out.nnn/
Zone X outbound directory
domain.nnn/
Domain/Zone X outbound directory
tick/
Tick (.tic) hold directory

/usr/lib/fidogate/
Main FIDOGATE library directory:
bin/
Sub-directory for executable binaries and scripts.
